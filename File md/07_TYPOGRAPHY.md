# 07_TYPOGRAPHY.md

# Typography System

## Philosophy

Typography is the interface.

Content should be readable before it is decorative.

---

## Objectives

- Create trust
- Improve readability
- Establish hierarchy
- Guide attention
- Reduce cognitive load

---

# Font Selection

Choose at most two font families.

Preferred combinations:

- Geist + Geist Mono
- Inter + JetBrains Mono
- IBM Plex Sans + IBM Plex Mono
- Manrope + JetBrains Mono

Avoid mixing more than two families.

---

# Heading Scale

H1
- One per page
- Primary statement
- Strong visual weight

H2
- Major sections

H3
- Subsections

H4–H6
- Supporting hierarchy only

Never skip heading levels.

---

# Paragraph Rules

- Comfortable line length (60–80 characters)
- Consistent spacing
- Avoid large text blocks
- Prefer short paragraphs

---

# Visual Hierarchy

Primary
- Hero title
- Important metrics
- Section headings

Secondary
- Descriptions
- Labels
- Navigation

Tertiary
- Metadata
- Timestamps
- Helper text

---

# Alignment

Prefer left alignment.

Avoid centered paragraphs except hero statements.

---

# Emphasis

Use emphasis sparingly.

Priority:
1. Weight
2. Size
3. Color
4. Decoration

Never rely on color alone.

---

# Spacing

Keep consistent rhythm.

Examples:

Heading → Paragraph
24–32px

Paragraph → Paragraph
16–24px

Section → Section
64–120px

---

# Accessibility

Minimum body size: 16px.

Maintain sufficient contrast.

Never sacrifice readability for aesthetics.

---

# Anti Patterns

Do NOT:

- Use multiple decorative fonts
- Random font sizes
- Inconsistent line heights
- Tiny body text
- Excessive uppercase
- Justified paragraphs

---

# Goal

Typography should communicate professionalism, clarity, and confidence before any animation or visual effect.

End of Typography Guide.
