# SEO Guide

## Philosophy

Every page should be discoverable, understandable, and indexable.

SEO starts with content, not keywords.

---

# Metadata

Every page must include:

- Title
- Meta description
- Open Graph title
- Open Graph description
- Open Graph image
- Canonical URL

Avoid duplicate metadata.

---

# Headings

Use:

- One H1 per page
- Logical H2 hierarchy
- H3-H6 only when needed

Never skip heading levels.

---

# URLs

Prefer:

/about

/services

/articles/legal-opinion

Avoid long or unclear URLs.

---

# Content

Write for humans first.

Use:

- Clear language
- Helpful information
- Descriptive headings

Avoid keyword stuffing.

---

# Images

Every image should have:

- Alt text
- Optimized size
- Modern format

Use descriptive file names.

---

# Structured Data

Use Schema.org when appropriate.

Examples:

- Person
- Organization
- Article
- FAQ
- Breadcrumb

---

# Internal Linking

Link related pages together.

Help users and search engines discover content.

---

# Technical SEO

Generate:

- sitemap.xml
- robots.txt

Avoid broken links.

---

# Performance

Fast websites rank better.

Optimize:

- Images
- Fonts
- JavaScript
- CSS

---

# Checklist

Before publishing:

[ ] Unique title
[ ] Meta description
[ ] Canonical URL
[ ] Open Graph
[ ] Semantic headings
[ ] Alt text
[ ] Sitemap
[ ] Robots
[ ] Internal links
[ ] Optimized assets

End of SEO Guide.
