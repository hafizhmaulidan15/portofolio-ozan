# Motion Design System

## Philosophy

Motion exists to communicate.

Never animate for decoration alone.

Every animation should help users understand:

- What changed
- Where it changed
- Why it changed
- What they can do next

---

# Principles

- Fast
- Natural
- Consistent
- Subtle
- Purposeful

Motion should never become the main attraction.

---

# Duration

Micro interactions
100–180ms

Buttons
150–200ms

Cards
180–240ms

Dialogs
220–320ms

Page transitions
250–400ms

Avoid animations longer than 500ms unless storytelling requires it.

---

# Easing

Preferred:

- ease-out
- ease-in-out

Avoid:

- Excessive bounce
- Elastic animations
- Cartoon effects

---

# Hover States

Allowed:

- Slight elevation
- Soft shadow increase
- Accent color transition
- Scale up to 1.02

Avoid:

- Large zoom
- Rotation
- Flashing colors

---

# Loading States

Prefer:

- Skeleton loaders
- Progress indicators
- Optimistic UI

Avoid:

- Infinite spinners without feedback

---

# Scroll Motion

Use scroll animation only to:

- Reveal content
- Guide reading flow
- Highlight section changes

Do not animate every element independently.

---

# Micro Interactions

Include subtle feedback for:

- Buttons
- Links
- Form fields
- Toggles
- Navigation
- Tabs
- Cards

Every interaction should acknowledge user input.

---

# Page Transitions

Transitions should preserve context.

Users should feel continuity rather than interruption.

---

# Accessibility

Respect reduced-motion preferences.

Provide meaningful experience even when animations are disabled.

---

# Anti Patterns

Do NOT use:

- Constant floating
- Random bouncing
- Spinning icons
- Parallax overload
- Auto-playing decorative motion
- Excessive blur transitions

---

# Motion Quality Checklist

Before delivery verify:

- Motion supports usability
- Timing is consistent
- No distracting animations
- Keyboard interactions remain smooth
- Performance remains high

---

# Goal

Motion should make the interface feel polished, responsive, and professional without distracting from the content.

End of Motion Guide.
