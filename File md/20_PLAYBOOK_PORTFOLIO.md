# 20_PLAYBOOK_PORTFOLIO.md

# Portfolio Playbook

## Objective

Create a portfolio that builds trust, communicates expertise, and guides visitors toward making contact.

---

# Architecture

Project Type:
Portfolio

Preferred Stack:
- Next.js
- TypeScript
- Tailwind CSS
- Framer Motion (optional)
- Static content
- No backend by default

---

# Recommended Sitemap

/
├── Home
├── About
├── Experience
├── Projects
├── Skills
├── Certifications
├── Achievements
├── Contact

Optional:
- Articles
- Gallery
- Resume

---

# Hero Section

Must answer:

- Who are you?
- What do you do?
- Why should visitors care?

Include:

- Name
- Professional title
- Short value proposition
- Primary CTA
- Secondary CTA

---

# About

Include:

- Professional summary
- Background
- Values
- Interests
- Career goals

---

# Experience

For each experience:

- Organization
- Role
- Duration
- Responsibilities
- Outcomes

Focus on measurable impact.

---

# Projects

Each project should include:

- Problem
- Solution
- Technologies
- Challenges
- Results
- Repository (if available)
- Live demo (if available)

---

# Skills

Group by category:

- Technical
- Design
- Tools
- Soft Skills

Avoid long unordered lists.

---

# Contact

Include:

- Email
- LinkedIn
- GitHub (if available)
- CV download

Provide a clear call to action.

---

# Visual Style

- Strong typography
- Consistent spacing
- Editorial layout
- Minimal decoration
- Accessible colors
- Responsive design

---

# SEO

Every page should include:

- Metadata
- Open Graph
- Structured headings
- Optimized images

---

# Final Checklist

[ ] Clear value proposition
[ ] Strong CTA
[ ] Professional copy
[ ] Responsive
[ ] Accessible
[ ] SEO optimized
[ ] Fast loading
[ ] Production ready

End of Portfolio Playbook.
